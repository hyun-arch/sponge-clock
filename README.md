# 🧽 Sponge Clock — 발표용 타이머 (Chrome Extension)

긴급 발표에서 **3초 안에 시간 세팅 → 시작**. 그리고 **다른 창을 띄워도 화면 맨 앞에 계속 떠 있는** 미니 타이머.

**▶ 바로 열기 (어느 기기에서나):** https://hyun-arch.github.io/sponge-clock/

## 설치 방법 (개발자 모드 로드)

1. Chrome 주소창에 `chrome://extensions` 입력
2. 우측 상단 **개발자 모드(Developer mode)** 켜기
3. **압축해제된 확장 프로그램을 로드(Load unpacked)** 클릭 → 이 폴더(`sponge clock`) 선택
4. 툴바의 🧽 아이콘 클릭 → **팝업 컨트롤 센터**가 열림

> 업데이트했을 땐 `chrome://extensions`에서 새로고침(↻) 한 번 눌러주세요.

## 4가지로 띄울 수 있어요

팝업(툴바 아이콘)에서 원하는 방식을 고릅니다.

| 방식 | 설명 |
|------|------|
| **팝업** | 툴바 아이콘 클릭 → 빠른 프리셋·시작·정지 컨트롤. 잠깐 확인용 |
| **미니창 (항상 위로)** | 화면 **우측 상단 구석**에 뜨는 작은 창. 안의 `📌` 버튼을 누르면 **모든 창·앱 위에** 고정되어 발표 슬라이드 위에서도 계속 보임 (Picture-in-Picture) |
| **전체화면** | 발표 모드. 링만 거대하게, 컨트롤은 숨김 |
| **새 탭** | 큰 화면으로 보기 |

### 미니창을 항상 위로 (핵심)
미니창 안의 **📌 버튼** → 진짜 항상-위 플로팅 타이머로 전환됩니다. 슬라이드를 전체화면으로 띄우거나 다른 앱을 써도 구석에서 계속 보입니다. 해제하려면 다시 📌 또는 “항상 위 해제”.

### 자동 실행
팝업 하단 **“브라우저 시작 시 미니창 자동 실행”** 을 켜두면, 크롬을 켤 때마다 미니 타이머가 자동으로 구석에 떠 있습니다.

## 단축키 (전체화면·새 탭 화면)

| 동작 | 키 |
|------|----|
| 시작 / 일시정지 | `Space` |
| 리셋 | `R` |
| 전체화면 | `F` |
| 항상 위로 미니 열기 | `P` |
| 설정 (테마·색상·음성·점멸) | `S` |
| 다음 구간 (어젠다) | `N` |
| 1분 추가 | `+` |
| 소리 끄기/켜기 | `M` |

## 특징

- **모든 창이 하나의 상태를 공유** — 팝업에서 시작해도 미니창·전체화면이 똑같이 동기화 (chrome.storage + 타임스탬프 기반이라 항상 정확)
- **카운트다운 / 스톱워치** 두 모드, 프리셋 1~60분 + 직접 입력
- **다구간 어젠다 타이머**: "발표 20분 → Q&A 5분"처럼 구간을 이어 실행. 구간이 끝나면 자동으로 다음으로 넘어가며 차임·음성으로 전환 안내. 화면엔 현재 구간 이름·`1/3`·"다음: Q&A"·전체 남은 시간 표시. `N`(또는 ⏭ 버튼)으로 즉시 다음 구간. 직접 입력 대화상자의 **어젠다** 탭에서 구간 추가/삭제·프리셋(발표+Q&A 등)
- **종료 시각 모드**: "오후 3시까지"처럼 절대 시각 입력 → 그 시각에 0이 되도록 자동 카운트다운(이미 지난 시각이면 내일 기준). 직접 입력 대화상자의 **종료 시각** 탭
- **색상 존**: 여유(파랑) → 절반(초록) → 경고 20%(주황) → 위험 10%(빨강) → **초과 시 `+MM:SS`**
- **사운드 알림**: 1분 전 경고음 · 마지막 10초 틱 · 종료 차임 (보이는 창에서만 울려 중복 없음)
- **음성 안내(TTS)**: "1분 남았습니다 / 30초 남았습니다 / 시간이 종료되었습니다" 음성. 설정에서 켜고 끄기 (음성 켜면 1분·종료는 음성이 대신, 10초 카운트는 비프음 유지)
- **테마 / 강조색**: 다크·라이트 + 강조색 5종(블루·퍼플·그린·핑크·그래파이트). 경고(주황)·위험(빨강) 색은 그대로 유지돼 긴급도는 항상 동일하게 읽힘
- **초과 시 화면 점멸**: 시간을 넘기면 화면 가장자리가 빨갛게 점멸 (설정에서 끄기 가능)
- **백그라운드 알림**: 모든 창이 가려져 있어도 시간이 끝나면 시스템 알림으로 알려줌

## 확장 없이 어디서나 쓰기 (웹 / PWA)

크롬 확장으로 설치하지 않아도, **그냥 웹페이지로** 똑같이 동작합니다. 환경을 자동 감지해서 확장이면 `chrome.storage`, 아니면 `localStorage`로 알아서 전환돼요.

- **로컬에서 바로**: `index.html`(또는 `timer.html`)을 더블클릭 → 브라우저에서 실행
- **어디서나 접속**: 이 폴더를 아무 정적 호스팅(예: GitHub Pages, Netflify, Vercel, 사내 서버)에 올리면 URL로 접속
- **앱으로 설치 (PWA)**: 호스팅한 주소에서 주소창의 **설치** 아이콘 → 독립 창 앱으로 설치. **오프라인에서도 동작**(서비스 워커가 캐시)
- **항상 위로**: 웹 버전에서도 `P` 또는 📌 버튼 → 현재 페이지가 Picture-in-Picture로 떠서 다른 창 위에 고정
- **화면 꺼짐 방지**: 카운트다운 중에는 Screen Wake Lock으로 화면이 자동으로 안 꺼짐

> PWA 설치·오프라인·항상위(PiP)는 `https` 또는 `localhost`에서 가장 잘 동작합니다. `file://` 더블클릭으로도 타이머 자체는 잘 돌아갑니다.

## 파일 구조

- `engine.js` — 공유 타이머 상태 머신 (storage 기반)
- `background.js` — 창 열기 / 알람 / 자동 실행 (서비스 워커)
- `popup.*` — 팝업 컨트롤 센터
- `mini.*` — 미니 플로팅 + 항상-위(PiP)
- `timer.*` — 전체화면 / 새 탭 발표 화면
- `sound.js`, `cues.js` — 사운드 알림
